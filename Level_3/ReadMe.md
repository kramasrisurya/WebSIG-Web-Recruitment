Flexbox
Flexbox is a CSS layout mode that makes it easy to align items in rows or columns. It automatically handles the size and spacing of elements so they fit perfectly within their container, even on different screen sizes.

CSS is Hard
CSS is tricky because small changes can have unexpected side effects across a webpage. Understanding how different properties—like margins, padding, and positioning—interact with each other takes a lot of trial and error to master.

The "Click" Moment
Everything clicked once I understood the Main Axis vs. the Cross Axis. Knowing that justify-content controls the horizontal flow and align-items controls the vertical flow made positioning elements feel simple instead of random.
